# Kairesa Website

Upload the contents of this folder to the root of the GitHub Pages repository. Keep `CNAME` in the repository root.

## Edit contact details
Search for `hello@kairesa.com` in `index.html`, `legal.html`, and `script.js`.

## Contact form
The form opens the visitor's email application. This works on GitHub Pages without a backend and does not store form data.

## Main files
- `index.html`: content and page structure
- `styles.css`: design and responsive behavior
- `script.js`: mobile navigation, theme switching, and contact form
- `assets/`: logo and social assets
